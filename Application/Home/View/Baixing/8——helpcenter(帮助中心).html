如何修改登录和交易密码 Q

A通过点击“用户中心”→“个人资料”中修改账户的登录密码及交易密码。

如何找回登录和交易密码 Q

A1、如果您是遗忘了登录密码，可以在登录界面点击“忘记密码”通过您的注册手机号来重新设置登录密码。
2、如果您是遗忘了交易密码，可以点击“用户中心”→“个人资料”→“交易密码”中的 “忘记交易密码”，同样通过您的注册手机号来重新设置平台交易密码。

如何进行身份认证 Q

A进入用户中心首页，点击实名认证标志进入认证页面，填写个人信息进行验证。

为什么实名认证无法通过 Q

A1、请确保您提供的姓名和身份证号与身份证上的一致。
2、请确认您是否曾经注册过百姓贷账号，并进行了身份认证。若不确定，请在客服中心留言或者拨打客服电话400-968-1688（工作日9:00-18:00）。
3、您的实名不能通过，可能有以下几点原因： 1）名字中带有繁体字 2）曾经改过名字 3）军人转业、复员换的身份证 4）做过户籍变动。

如何进行企业用户认证 Q

A企业用户需要提供企业三证照片，请事先按要求准备，以提高操作效率。企业用户需要提供企业三证（营业执照、组织代码证和税务登记证）照片，请事先按要求准备，以提高操作效率。 本站对于证明性资料只接收原件拍照上传，必须保留exif信息，不得做除调整大小之外的任何修改。

如何保证隐私和资料安全 Q

A平台非常重视用户的隐私和数据安全，我们采取国际标准的SSL安全加密技术保障数据传输安全。
保护用户的隐私和数据安全是我们最重要的责任之一，平台会严格按照隐私条款保护包括借入者和借出者在内的全部注册用户的信息。
前台显示的只是账户名等虚拟的信息，只要遵守借款协议，按时还款，就不会曝光资料，绝对安全。

平台的资金是否安全 Q

A用户充值和提现的资金的收付行为全部在具有央行颁发支付牌照的知名第三方支付平台（先锋支付、富友支付）以及银行体系内操作，相应的资金亦实际存放在这些机构的银行帐户内。

充值细则 Q

A步骤1：注册百姓贷平台账号，并且完成实名认证与手机认证。
步骤2：登录百姓贷账户，点击“首页导航”——“我的账户”——“充值”。
步骤3：输入充值金额，选择快捷支付或网银支付，点击“充值”，跳入第三方支付页面完成充值。<

银行已扣款，账户余额未显示 Q

A1、银行已扣款，但是可能由于银行系统原因，资金还未同步到第三方支付平台。此种情况，资金同步完成后余额即可变更。若银行未能把钱同步到第三方支付平台，银行会自动退款到充值银行卡。
2、第三方支付平台判断您的网络存在钓鱼风险，自动关闭交易，您的资金会自动退回您的充值银行卡。
3、请联系在线客服人员提供您的注册手机号码、充值金额、充值时间、充值订单号，我方为您查询，并告知您具体的结果。

提现细则 Q

A步骤1：注册百姓贷平台账号，并且完成实名认证与手机认证，同时绑定银行卡且必须同卡进出，如要修改提现卡号，请联系客服。
步骤2：登录百姓贷账户，点击“首页导航”——“我的账户”——“提现”。
步骤3：输入提现金额、平台交易密码、验证码，点击“确认提现”，完成提现。
（1）提现时间 : 7*24小时即时提现
（2）提现费用: 单笔提现金额允许范围为2元以上-50万元以下，每笔提现平台收取2元/笔提现费，同一账户每天提现总额无限制。提现银行账户需与实名认证身份一致。

投资抵用红包使用的规则 Q

A投资红包用于在投资时抵扣投资金额。每笔投资只能使用一个红包。
除指定情况下，红包使用比例为年化投资金额的1%。
如投资12月期限的投资计划10,000元，即年化投资额为10,000元，最高可使用100元抵用券，实际只需支付9,900元。

积分是什么，积分商城是什么 Q

A积分服务是百姓贷平台回馈广大投资人的增值服务之一。
积分达到一定数量，可兑换各类实物奖励。积分获取单位最小为1。
积分商城是平台基于积分回馈计划，面向所有用户提供的统一积分服务和消费应用。

如何获取积分 Q

A1、积分获取的比例为投资金额的0.02%，即投资5000元即可获得1点积分。投资后即刻到达积分账户。
例：当投资金额为100万时，相应的积分为200点。
2、其他活动获得积分，具体查看“用户中心——致富之路——其他奖励”查看。

积分的时效性 Q

A每一年的12月31日23点59分，系统会清理会员上一年所获得的积分。
例如：系统于2015年12月31日23点59分清理用户在2014年1月1日 - 2014年12月31日所获得的所有积分。

投标前需要注意的事项 Q

A1、投标前认真阅读该笔借款的详细信息（包括借款金额，利息率，借款期限、借款者信用等级等），以确定您所要投的标符合您的风险承受能力和所要求的投资回报率。
2、投标前应知道若您所投标的借款人发生违约，违约损失由投资该标的所有理财人共同承担。
3、投标前请核实自己将要理财的金额，确认无误后再进行操作。

如何查看投标中和投标成功的记录 Q

A投标后要等待对方满标，满标后3个工作日会有审核结果。
已经投标并审核通过的：依次“我的帐户”—“致富之路”—“投资项目”，此页面可分别查看待收款、收款中款项、已还清款项。

投标后是否可以撤回 Q

A投资者一旦投标成功后，投标资金不可以撤回。

投标时为什么资金会冻结 Q

A借出者对借款列表进行投标后，在此列表没有结束之前，这部分金额将被锁定。
如果列表成功，这部分金额将转给借入者；如果列表没有成功，这部分金额将立刻返还给借出者可以进行其他投标。

投标失败的原因 Q

A借出者对于投标失败的原因，主要是因为：借入者借款列表被审核失败。

等额本息还款法（每月还款）是什么意思 Q

A1、定义：等额本息还款法是每月以相等的还本付息数额偿还贷款本息。即借入者每月按相等的金额偿还贷款本息，其中每月贷款利息按月初剩余贷款本金计算并逐月结清。
2、由于每月的还款额相等，因此，在贷款初期每月的还款中，剔除按月结清的利息后，所还的贷款本金就较少；而在贷款后期因贷款本金不断减少、每月的还款额中贷款利息也不断减少，每月所还的贷款本金就较多。
3、其计算公式如下：
还款总金额=贷款本金×[月利率×（1+月利率）^还款月数]
每月还款额=还款总金额÷[（1+月利率）^还款月数—1]

如何还款 Q

A先充值到平台账户上，进入“用户中心”——“致富之路”——“借款管理”——“还款明细”点击“我要还款”，进行还款。

流标后是否还能发布借款 Q

A流标后可以根据系统提示的时间再次发布借款。

为什么我不能发布借款 Q

A1、资料未填写完整。
2、借款额度不够：您只能发布在自己借款额度内的借款列表，若想发布更大额的借款可以通过客服中心-额度相关提交问题，工作人员会根据您的实际情况提高额度。

如何发布借款列表 Q

A发布借款列表的流程：
“我要借款”—“选择要申请的标种”-填写“基本资料”-“申请额度”-“上传认证材料”，额度审批通过，即可开始发布借款。
